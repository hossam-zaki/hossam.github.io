What have I been doing?

Apollo - The Speech-to-Text Note Taking App

Apollo is an application where Doctors can take notes without ever having to touch their computer. We used Google Speech Recognition to transcribe visits and the Knuth-Morris-Pratt algorithm to extract important keywords. Fully encrypted using Google Tink. Also we conducted user tests with medical students at Brown Med, and doctors at Rhode Island Hospital

Computational Analysis of Sea Urchin Transcriptome

Using data from the Sea Urchin Transcriptome, I developed pipelines to search for motifs in genetic sequences using the Knuth-Morris-Pratt algorithm and global/local alignment. I am also building a machine learning model to find more accurate and motifs using CHIP-Seq data.

X-Ray Classification

Using X-Ray data from the NIH and COVID-19 X-Rays from NCBI, I built a neural network which achieved an average of 65% false positive rate across all of the labels using a simplistic model of only a few layers

Courier

In light of COVID-19, we wanted to connect neighbors together when they need each other the most. Using this app, neighbors can request and accept services for their fellow neighbors. Used a KDTree to find K-Nearest-Neighbors. Built using Python, Flask, CSS, MongoDB, and Google Geolocation API

College Boss

Developed a website and online services to help students succeed in the college admissions process. Developed using Python, Flask, Bootstrap, Node.js, and Stripe

COVID-19 Sequence Similarity

This project was built to compare the sequence of COVID-19 with several other viruses. We used Global and Local alignment to generate a similarity score, then used Knuth-Morris-Pratt to test several motifs in these sequences

Protein Structure Similarity

This project was during my internship at the Protein Data Bank of Europe. I developed a pipeline to compare different protein structures in a particular protein family. I used Python, C++, SQL, and PyMol for the superposition of the structures, then hierarchical clustering techniques to detect protein similarity

Structural Variant Characterization in tumorigenesis

This project was done during my time at the Broad Institute of Harvard and MIT. Using data from ICGC, PCAWG, TCGA, Ensembl, and Uniprot, I characterized structural variants in DNA Repair Genes involved in tumorigenesis. Utilized Python, R, and Sklearn

Real Time Voting App

In Google's Software Product Sprint, myself and 3 other developers built a real-time voting app using Java, Google Cloud, and HTML/CSS/JS. Built NLP Algorithm to find similar questions on quora
